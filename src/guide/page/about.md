## 添加

1. 前往你的 Hexo 博客的根目录

2. 输入 `hexo new page about`

3. 你会找到 `source/about/index.md` 这个文件

4. 修改这个文件：

   记得添加` type: "essay"`

   ```markdown
   ---
   title: 关于
   date: 2022-06-07 22:17:49
   type: "about"
   aside: false
   ---
   ```
